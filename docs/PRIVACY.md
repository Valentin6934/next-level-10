# Confidentialité V1

Les données sont enregistrées dans le navigateur de l’appareil.

Elles ne sont pas envoyées à Next Level 10, car la V1 n’a pas de serveur utilisateur.
L’export JSON est déclenché volontairement par le joueur.
La suppression des données efface le stockage local de l’application.

Avant une future ouverture publique, il faudra ajouter :
- consentement parental adapté aux mineurs ;
- politique de confidentialité complète ;
- sécurité des comptes ;
- chiffrement des données sensibles ;
- procédure de suppression de compte.
