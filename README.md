# Next Level 10 V1.6.0 — Sprint 3.3 final

Application PWA football local-first avec NOVA, suivi de récupération, nutrition, objectifs quotidiens, badges, statistiques et fonctionnement hors ligne.

## Lancer localement

Servez le dossier avec un serveur HTTP, par exemple `python3 -m http.server 8080`, puis ouvrez `http://localhost:8080`.
